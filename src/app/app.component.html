Question is; How do we keep multiple tab in sync? With ngxs, localstorage and rxjs its possible.
Tool;

  • Ngxs for state management (Best state management library + pattern for Angular projects)
  • localStorage for keeping actions
  • Rxjs for detecting changes on window.storage with fromEvent operator

Steps;

  1. We're going to build some delightful form with ngxsForm
  2. We re going to save our ACTION to localstorage
  3. Listen to window.storage event with fromEvent operator

Result;

  1. We can keep sync multiple tabs. We will change data in one tab, and another tab will update automatically.
